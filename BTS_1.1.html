My First Bootstrap Page

1) A cosa serve bootstrap?

Bootstrap è un framework front-end gratuito per uno sviluppo Web più rapido e semplice

2) Cosa devi importare per creare una pagina web con bootstrap

<link rel="stylesheet" href="https://maxcdn.bootstrapcdn.com/bootstrap/4.3.1/css/bootstrap.min.css">

3) Cos’è un CDN? Che vantaggi comporta rispetto a scaricare bootstrap sul tuo sito?

CDN (Content Delivery Network) MaxCDN fornisce supporto CDN per CSS e JavaScript di Bootstrap. È inoltre necessario includere jQuery
I vantaggi che porta sono che porta tempi di caricamento più rapidi. CDN assicurerà che una volta che un utente richiede un file da esso, verrà servito dal server più vicino a loro e i tempi di caricamento saranno più rapidi.

4) A cosa serve questo attributo del tag meta? content="width=device-width, initial-scale=1

"width=device-width. Imposta la larghezza della pagina per seguire la larghezza dello schermo del dispositivo (che varierà a seconda del dispositivo).
initial-scale=1. Imposta il livello di zoom iniziale quando la pagina viene caricata per la prima volta dal browser
